## Функции в `contacts.py`

### 1. `add_contact(contacts, name, phone, email)` 
Добавляет новый контакт в список.
- **Параметры**: список контактов, имя, телефон, email
- **Возвращает**: True если успешно, False если имя пустое

### 2. `show_all_contacts(contacts)`
Возвращает список всех контактов в виде строк.
- **Параметры**: список контактов
- **Возвращает**: список строк с форматированными контактами

### 3. `find_contact(contacts, name)`
Ищет контакт по имени (без учета регистра).
- **Параметры**: список контактов, имя для поиска
- **Возвращает**: словарь контакта или None

### 4. `find_all_by_name(contacts, name)`
Ищет все контакты, содержащие имя (частичный поиск).
- **Параметры**: список контактов, имя для поиска
- **Возвращает**: список найденных контактов

### 5. `delete_contact(contacts, name)`
Удаляет контакт по имени (без учета регистра).
- **Параметры**: список контактов, имя контакта
- **Возвращает**: True если удален, False если не найден

### 6. `get_contact_by_phone(contacts, phone)`
Ищет контакт по номеру телефона.
- **Параметры**: список контактов, номер телефона
- **Возвращает**: словарь контакта или None

### 7. `update_contact(contacts, name, phone=None, email=None)`
Обновляет информацию контакта.
- **Параметры**: список контактов, имя, новый телефон (опционально), новый email (опционально)
- **Возвращает**: True если обновлен, False если не найден

### 8. `get_contacts_count(contacts)`
Возвращает количество контактов.
- **Параметры**: список контактов
- **Возвращает**: целое число - количество контактов

### 9. `clear_all_contacts(contacts)`
Удаляет все контакты.
- **Параметры**: список контактов
- **Возвращает**: None
